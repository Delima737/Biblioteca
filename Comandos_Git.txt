git add "FILENAME.TXT" = Adiciona o arquivo na area de estagios
A area de estágio é a parte 2 do processo de Commit no Git; A area de estágio
 é como se fosse a sala de espera para as suas mudanças no arquivo até você dar um
 commit nelas.


git status = Ver como está o estado geral do git, após dar um git add em
um arquivo, verá que o nome dele se tornará verde, usando git status,
o que significa que agora ele está na area de estagio.



git commit = Ele leva as mudanças de um ambiente local para o repositório no git,
 permitindo ainda a inserção de uma mensagem descritiva. Assim, a cada mudança ou
 finalização de uma tarefa, a pessoa desenvolvedora pode submeter seus feitos e deixar
 claro para as outras pessoas o que ela fez.
Para saber algumas informações sobre a ramificação na qual você está trabalhando agora, use o “status”. Esse comando esclarece quais arquivos foram alterados e faz uma comparação com relação à ramificação principal.

git commit -m "Comentario" = deixa um comentario descritivo da mudança.
